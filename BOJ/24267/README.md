i <- 1 to n-2, j <- i+1 to n-1, k <- j+1 to n

n = 7 일때 i는 1 to 5이다.
i = 1일때 j는 2 to 6, i = 2일때 j는 3 to 6, i = 3일때 j는 4 to 6이다.
i = 1, j = 2이면 k는 3 to 7이고, j = 3이면 k는 4 to 7, j = 4이면 k 5 to 7이다.

i = 1, j = 2이면 k는 3이상, j = 3이면 k는 4이상이다.
k는 항상 j+1부터 시작하므로 k>j, j는 i+1부터 시작하므로 j>i, i는 1부터 시작하므로 1 <= i < j < k <= n이다.
알고리즘의 수행 횟수는 이 부등식을 만족하는 i j k 의 갯수를 구하는 것이고, 즉 n개 중 3개를 순서없이 뽑는 경우의 수 (조합)이다.
(1,2,3) (1,2,4) (1,2,5) ... (1,6,7) (2,3,4) ... (5,6,7) 와 같이 돌게된다.

따라서 알고리즘의 수행 횟수는 nC3 = n(n-1)(n-2)/6 이고 시간복잡도는 O(n^3)이다.
