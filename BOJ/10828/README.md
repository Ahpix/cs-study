스택은 리스트만으로도 구현할 수 있지만, 자료구조를 잘 이해하기 위해 class로 직접 구현해보았다.
push, pop, top, size, empty 메서드를 분리하여 함수 간 호출 구조를 만들었다.

pop에서 top이나 isEmpty를 호출하는 등 메서드 내에서 또 다른 메서드를 호출하는 구조는 복잡해보였고,
단순하고 가독성 높게 구현 할 수 있다는 것을 인지했다. 하지만 코드 최적화보다는 직접 구현 및 메서드를 사용하는 연습에 의미를 두었다.

입력을 input()으로 처리했을 때 시간초과가 발생하여 sys.stdin.readline()으로 변경하며 최적화가 중요하다는 것을 깨달았다.
